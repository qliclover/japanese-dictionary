# 日语字典

一个简洁的日语单词学习工具。

## 🚀 使用方法

### 在线使用
直接访问：**[https://qliclover.github.io/japanese-dictionary/](https://qliclover.github.io/japanese-dictionary/)**

### 本地使用
下载 `index.html` 文件，双击用浏览器打开。

---

## 📖 快速开始

### 1️⃣ 上传单词文件
1. 点击 **"上传文件"** 按钮
2. 选择要放入的文件夹（或选根目录）
3. 选择你的单词文件（支持 `.json`、`.xlsx`、`.xls`）
4. 可以下載github中的文件夾到本地然後上傳
5. 可以一次选择多个文件（按住 Ctrl/Cmd）

### 2️⃣ 浏览和搜索
- 点击文件夹查看单词
- 点击 **"📚 全部"** 查看所有单词
- 在搜索框输入关键词实时搜索

### 3️⃣ 收藏生词
- 浏览单词时点击 ☆ 收藏到生词本
- 点击 **"生词本"** 查看所有收藏
- 可以手动添加单词

### 4️⃣ 开始学习
- 选择一个文件夹
- 点击 **"开始学习"**
- 每次随机学习20个单词
- 用空格键显示答案，方向键切换

---

**学习模式：**
- `空格` - 显示/隐藏答案
- `←` / `→` - 上一个/下一个单词
- `Esc` - 退出

---


## ❓ 格式

支持 `.json`、`.xlsx`、`.xls` 格式。
